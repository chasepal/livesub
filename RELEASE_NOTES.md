# Release Notes

## v0.2.0-alpha

First GitHub-ready alpha under the LiveSub name.

Highlights:

- Renamed public app identity to `LiveSub`
- New bundle ID: `app.livesub.mac`
- Release build packaging by default
- GitHub zip artifact: `LiveSub-v0.2.0-alpha-macos.zip`
- Text-only subtitle overlay is the default mode
- Default subtitle placement is bottom-centered
- Cloud AST mode remains the recommended low-resource mode
- Root README, Chinese README, privacy note, roadmap, and MIT license added

Known limitations:

- App is ad-hoc signed, not notarized
- Chrome capture is app-level, not tab-level
- First-run onboarding is still basic
- Screenshots and demo GIF are not yet included
